This section will describe the core functionalities that in clustimage. Many of the functionalities are written in a generic manner to make sure that it can be used in various applications.

PreprocessingÔÉÅ

The functions that handle the preprocessing of images is concentrated into the function clustimage.clustimage.Clustimage.imread(). This function reads the full path to the image that needs to be imported, and is subsequently colour-scaled (gray/rgb), scaled and resized.

The pre-processing has 4 steps and are exectued in this order.

    1. Import data.

    1. Scaling color pixels between [0-255]

    1. Conversion to gray-scale (user defined)

    1. Resizing (to reduce computation time)

# Import library
import matplotlib.pyplot as plt
import clustimage as cl
# Load example dataset
path_to_imgs = cl.import_example(data='flowers')
# Run specific function to read preproces the images
img = cl.imread(path_to_imgs[0], dim=(128,128))
# Plot the flower after preprocessing
plt.figure()
plt.imshow(img.reshape(128,128,3))
plt.axis('off')
Left is orignal input figure and right is after preprocessingÔÉÅ

ColorscaleÔÉÅ

The img_read function clustimage.clustimage.img_read() reads the images and colour scales it based on the input parameter grayscale. This function depends on functionalities from python-opencv and uses the cv2.COLOR_GRAY2RGB setting.

imscaleÔÉÅ

The scale function clustimage.clustimage.img_scale() is only applicable for 2D-arrays (images). Scaling data is an import pre-processing step to make sure all data is ranged between the minimum and maximum range.

The images are scaled between [0-255] by the following equation:

Ximg * (255 / max(Ximg) )

imresizeÔÉÅ

The resize function clustimage.clustimage.img_resize() resizes the images into 128x128 pixels (default) or an user-defined size. The function depends on the functionality of python-opencv with the interpolation: interpolation=cv2.INTER_AREA.
